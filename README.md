# ban-free-api

这里并不是任何一个厂商的free，🤗这里提出一种可能的提高逆向成本的方案，希望厂商们让free-api早日终结👋

[free-api系列项目](https://github.com/orgs/LLM-Red-Team/repositories?q=free-api) 可预见的结果是沦为号商的玩物，并不能普惠大众，厂商们如果能够自行解决风险来减少逆向，从而抵制号商将让生态变好一些，避免让劣币驱逐良币。
